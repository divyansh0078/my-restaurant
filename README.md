Follow these steps to get the system running locally:

### 1. Backend Setup

1.  Navigate into the backend directory:
    ```bash
    cd backend
    ```
2.  Install dependencies (if using Node.js or similar):
    ```bash
    npm install
    ```

3.  Set up environment variables (database connection string, port, etc.) in a file named `.env`.

    ```bash
    npm start 
    ```

1.  Open a new terminal and navigate into the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install 
    ```
3.  Start the client application:
    ```bash
    npm start 
    ```
